# 🎓 智慧手表 STM32 仿真项目

## 📝 项目简介
这是一个基于 STM32F103C8T6 主控芯片的智慧手表仿真项目。本项目使用 Keil V5 进行代码编写，并使用 Proteus 软件进行硬件电路仿真，旨在模拟智能穿戴设备的核心功能。

## ✨ 主要功能
- ⏰ **实时时钟**：精准的时间显示与系统运行。
- ⏰ **闹钟设置**：支持用户自定义闹钟时间并进行提醒。
- 🌡️ **温湿度采集**：实时读取并显示环境温湿度数据。
- 🚨 **阈值警报**：当温湿度超出预设的安全阈值时，系统会自动触发警报提示。

## 🛠️ 开发环境
- 主控芯片：STM32F103C8T6
- 编译器：Keil MDK-ARM V5
- 仿真软件：Proteus 8.x

## 🚀 如何运行项目（使用指南）
1. 下载本仓库的代码到电脑。
2. 双击打开 `Keil工程文件.uvprojx`，点击编译（Build），确保成功生成 `.hex` 文件。
3. 打开 Proteus 仿真文件 `.pdsprj`。
4. 双击原理图中的 STM32 芯片，在 "Program File" 中选择刚才生成的 `.hex` 文件。
5. 点击 Proteus 左下角的运行按钮，即可开始仿真测试！

## 📂 目录说明
- `watch/`：存放 Keil 工程文件
- `Proteus/`：存放 Proteus 仿真原理图
- `MDK-ARM/`：核心业务逻辑代码
- `task/`：外设驱动代码
